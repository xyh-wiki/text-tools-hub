# Text Tools Hub

A pure front-end React + Vite project that provides a collection of client-side text and code utilities.
All tools run in the browser, with no backend or SaaS features.

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Tech stack

- React 18
- TypeScript
- Vite
- react-router-dom
- react-helmet-async

## Available tools (MVP)

- Text tools: remove duplicate lines, remove extra spaces, case converter
- Formatters: JSON formatter (with minify)
- Converters: Base64 encode / decode
- Regex tools: email extractor, URL extractor, remove special characters
- Random: password generator, UUID generator
